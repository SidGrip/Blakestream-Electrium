# Coin Overlays

Each subdirectory here is reserved for one coin-specific Electrium overlay.

Planned ownership per overlay:

- wallet/app name
- ticker and display units
- address and Bech32 constants
- URI scheme
- explorer defaults
- icons and packaging assets

Current reserved overlays:

- `BLC`
- `BBTC`
- `ELT`
- `LIT`
- `PHO`
- `UMO`
